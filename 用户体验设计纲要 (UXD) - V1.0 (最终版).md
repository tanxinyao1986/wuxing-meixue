# 用户体验设计纲要 (UXD) - V1.0 (最终版)

- **文档ID:** UXD-MVP-V1.0
- **版本:** 1.0 (最终版)
- **关联产品:** 五行节律 / Daily Verse
- 

---

## 1. 核心设计原则

1. **简洁至上:** 每个界面只做一件事。
2. **正向引导:** 永远使用积极、温暖、鼓励的语言。
3. **美学优先:** 追求极致的“苹果”美学，视觉体验是产品核心价值。
4. **非侵入式:** 尊重用户，克制打扰。

---

## 2. 核心用户流程

1. 用户打开App → 进入**[今日]的[每日指南视图]**。
2. 用户左滑 → 过渡至**[明日]的[每日指南视图]**。
3. 用户点击“日历”图标 → 切换至**[月历视图]**。
4. 用户在月历上点选任一天 → 跳转至**该日的[每日指南视图]**。

---

## 3. 视觉与元素设计

### 3.1: 界面元素详解

- **模块图标:** 五个模块图标**必须**是体现五行元素的抽象符号（如**叶子、火苗、水滴、金属光泽、陶土质感**），并具备微妙的待机动画，赋予生命感。
- **字体 (Typography):** 标题建议使用优雅的衬线字体（如**宋体-简**），正文使用清晰的非衬线字体（如**苹方**），营造专业与美学的平衡。
- **质感与光效 (Texture & Effects):** 中央圆环及背景需使用**动态光效**和**细腻肌理**，提升高级感。展开的内容卡片**必须**使用苹果系统标准的**“毛玻璃/磨砂”效果(Frosted Glass)**。

### 3.2: 色彩 (Color Palette)

- **主色调:** 以深灰/近黑作为背景，或纯白作为背景，营造极致的对比和宁静感。
- **点缀色:** 当日五行对应的能量色（绿、红、黄、白、黑/蓝）将作为当日界面的主点缀色。

---

## 4. 交互与动效设计

### 4.1: 核心动效

- 点击模块图标，图标放大并“生长”出内容卡片，其他图标缩小退后。动画曲线**必须**使用“**Ease-in-out**”，时长建议为**0.4秒**，确保动画流畅、优雅、不突兀。

### 4.2: 触觉反馈 (Haptics)

- 在“**左右滑动切换日期**”、“**点击展开卡片**”、“**点击日历图标**”三个关键交互节点，**必须**加入**细腻(Subtle)**的触觉反馈(Haptic Feedback)。

---

## 5. 无障碍设计 (Accessibility)

- **动态字体:** **必须**支持iOS动态字体(Dynamic Type)，确保所有文本在用户调整系统字号后能正确缩放。
- **旁白 (VoiceOver):** 关键交互元素（如按钮、模块图标）**必须**提供清晰的VoiceOver标签。
